Step 2: Outline
You are a YouTube content strategist and prompt engineer.

Create a detailed YouTube video outline for the video titled:
“5 AI Tools Every College Student Must Use in 2025”

**Target audience:**
Indian college students (18–24), non-technical, are overwhelmed with studies.

**Video length:**
8–10 minutes.

**Requirements:**

- Start with a strong hook related to student pain points
- Short intro (who the video is for + what they’ll gain)
- 5 main sections (one per AI tool)
**For each tool include:**
    - Tool name
    - What problem it solves
    - How a student actually uses it
    - One real-life example
- Add a recap section
- End with a clear CTA (like, comment, subscribe)

**Output format:**
Clear headings + bullet points.
No script. Outline only.”
