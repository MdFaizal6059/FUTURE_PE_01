# FUTURE_PE_01 – AI-Powered YouTube Video

## Overview
This project was completed as Task 1 of the Prompt Engineering Internship
at Future Interns. The objective was to use prompt engineering techniques
to generate a complete YouTube video workflow including topic ideation,
script writing, thumbnail creation, and AI video production.

## Video Title
5 AI Tools Every College Student Must Use in 2025

## Tools Used
- ChatGPT (script & prompt generation)
- Canva / CapCut / Pictory (video creation)
- Canva (thumbnail design)

## Prompt Engineering Approach
Structured prompts were used at each stage to control:
- Audience targeting
- Tone and clarity
- Output structure
- Reduction of random or low-quality AI responses

## Deliverables
- Full YouTube script
- AI-generated thumbnail
- Final video (linked)
